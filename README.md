# 概要

タイポグリセミア現象を使用したサービスです。
入力された文章をタイポグリセミア変換します。

# タイポグリセミア現象とは

タイポグリセミア現象とは、文章中の単語の文字を並べ替えても、多くの人が正しく読めてしまう現象です。

# セットアップ

- リポジトリをクローン

```bash
$ git clone git@github.com:ishi720/make_typo.git
```

- 必要パッケージをインストール

```bash
$ yarn install
$ cd app & yarn install
```

- Gulpデプロイ

```bash
$ yarn gulp
```
